Usage Guidelines

✓ Do

  • • Use the logo with adequate spacing
  • • Use the accent blue for important CTAs
  • • Maintain consistent typography hierarchy
  • • Use the full logo on light backgrounds

✗ Don't

  • • Stretch or distort the logo
  • • Change the logo colors
  • • Use the logo on dark backgrounds without contrast
  • • Mix fonts inconsistently
