# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Reporting a Vulnerability
Please email security@kyron.so with:
- Description of the vulnerability
- Steps to reproduce
- Possible impact

You may use our PGP key (below) to encrypt reports. We will acknowledge within 24 h and keep you informed of remediation progress.
